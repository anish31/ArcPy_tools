Tool is used to extract a set of rasters using a mask and export the masked 
raster's attribute table to .csv file. The input to be provided is a folder 
containing raster dataset followed by shapefile (mask) used to extract the dataset from input 
raster.The folder where output i.e. extracted raster and .csv file is written 
to be provided.  

Note the raster should be .tif format. 

To use this tool copy the folder and place it any drive. 

Open the ArcGIS 10.x and locate this folder through Arc Catalog

Expand the CDL Clipper toolbox and double click on the CDL script.
